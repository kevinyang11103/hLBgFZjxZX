# 客户关系管理系统 python906

## 项目概述

客户关系管理系统是一款基于Python语言的软件，旨在帮助各类人员更高效地管理客户信息、销售订单和运营策略。系统采用Django框架进行后端开发，结合Vue框架构建友好的用户界面。

## 技术栈

- **后端**: Python语言，Django框架
- **前端**: Vue框架
- **数据库**: MySQL数据库
